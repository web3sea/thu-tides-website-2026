# Brevo API Documentation

Source: https://developers.brevo.com/docs/getting-started

## Core Capabilities

The Brevo platform offers programmatic access to all platform features through a unified REST interface. Key functionality includes:

- **Messaging**: Transactional and marketing communications via email, SMS, and WhatsApp with scheduling and tracking
- **Contact Management**: Audience building and targeted campaign execution
- **Sales & eCommerce**: Pipeline tracking and revenue attribution
- **Conversations**: Live chat integration with programmatic management
- **Custom Data**: Flexible objects and event tracking for specific needs

## API Solutions Available

The platform provides six primary API solutions:

1. **Messaging API** — Send transactional messages with batch operations and activity monitoring
2. **Account Management** — Configure senders, domains, users, and credentials programmatically
3. **eCommerce** — Sync product catalogs and orders; measure campaign performance
4. **Conversations** — Embed chat widgets and manage customer interactions
5. **Custom Objects** — Build and maintain specialized data structures
6. **Custom Events** — Monitor user behavior through JavaScript or REST endpoints
